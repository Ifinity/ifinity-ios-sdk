IFBackgroundGeosController Class Reference

Inherits from NSObject
Conforms to CLLocationManagerDelegate
Declared in IFBackgroundGeosController.h

Overview

This controller contains methods to handle collecting data in the background/foreground and do other stuff like notifications.

Discussion: This class helps to setup and handle all operation in background required to collect ibeacons stats. It requires some preparations in app, like turn on app background modes and requestAlwaysAuthorization in CLLocationManager. This class can also work as CLLocationManagerDelegate, otherwise calls from methods in CLLocationManagerDelegate have to be passed to this class.
