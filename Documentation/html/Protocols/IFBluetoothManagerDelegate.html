manager:isValidPeripheral:advertisementData:RSSI:

This method allow to ignore some peripherals. If method is implemented and returns NO, then peripheral is ignored and not process further.

- (BOOL)manager:(IFBluetoothManager *)manager isValidPeripheral:(CBPeripheral *)peripheral advertisementData:(NSDictionary *)advertisementData RSSI:(NSNumber *)RSSI

Parameters

manager

bluetooth manager

peripheral

peripheral to check

advertisementData

advertisement data recived in peripheral packet

RSSI

peripheral RSSI value

Return Value

If NO peripheral is ignored and not processed. If YES, peripheral is processed further and at some point pass to IFTransmitter.

Discussion

This method allow to ignore some peripherals. If method is implemented and returns NO, then peripheral is ignored and not process further.
